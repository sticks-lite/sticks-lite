# Code of Conduct

## Our Pledge

Sticks Lite is a learning-focused project for monitored educational
environments. We want teachers, students, maintainers, and contributors to feel
safe asking questions, reporting problems, and improving the language together.

We pledge to make participation in this project respectful, welcoming, and
constructive.

## Expected Behavior

- Be kind, patient, and specific.
- Assume good intent, especially with beginner questions.
- Give feedback about code, docs, and ideas without attacking people.
- Keep discussions appropriate for classroom and student audiences.
- Avoid sharing private student information, classroom records, access tokens,
  passwords, or other sensitive data.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Sexual, violent, hateful, discriminatory, or degrading language.
- Publishing private information without clear permission.
- Encouraging unsafe, illegal, or harmful use of Sticks Lite.
- Repeatedly disrupting issue discussions, pull requests, or maintainer work.

## Reporting

If you see behavior that violates this code of conduct, report it to the project
maintainers through GitHub. If the report includes sensitive information, do not
post it publicly in an issue.

Maintainers may remove comments, close issues, block users, or take other
reasonable action to protect the community and classroom users.

## Scope

This code of conduct applies to project spaces including GitHub issues, pull
requests, discussions, examples, documentation, and other Sticks Lite community
channels.
