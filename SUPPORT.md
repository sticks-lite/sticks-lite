# Support

Sticks Lite is an open source educational language. Support is community-based
and best-effort.

## Where To Get Help

- Use GitHub issues for reproducible bugs, documentation problems, feature
  requests, and classroom feedback.
- Check the README and docs before opening a new issue.
- Include small `.slite` examples when asking about language behavior.

## Good Support Requests

Helpful reports usually include:

- the Sticks Lite version,
- Node.js version,
- operating system,
- the exact command you ran,
- a minimal `.slite` source file,
- expected behavior,
- actual behavior.

## Classroom Feedback

Teachers, mentors, and students are welcome to share classroom feedback. Please
do not post names, grades, school records, private student work, contact
information, or other student-identifying information.

## Not Covered

The project cannot provide:

- emergency support,
- guaranteed response times,
- production deployment advice,
- legal, security, medical, financial, or safety-critical guidance,
- supervision for untrusted code execution.

Sticks Lite is intended for monitored educational environments and introductory
computer-science learning.
